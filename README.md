# AI-Air-Drawing-Pad

A starter project for an AI-powered air drawing pad.

## Setup

1. Create a virtual environment
2. Install dependencies from `requirements.txt`
3. Run `python main.py`
 
Running the demo (camera required):

```bash
python main.py --no-headless
```

Run headless (no camera loop):

```bash
python main.py
```

Run tests:

```bash
pip install -r requirements.txt
pytest -q
```
